# modules/

Esta carpeta es para módulos de terceros que se incluyen desde `description.ext` (por ejemplo los de GRAD).

Ejemplo de estructura cuando usas GRAD Persistence:

```
modules/
  grad-persistence/
  grad-fortifications/   (opcional)
```

> Nota: DON Framework no "vendorea" aquí dentro los repos de GRAD.
> Descárgalos desde GitHub ("Download ZIP") y copia su contenido dentro de estas carpetas.
